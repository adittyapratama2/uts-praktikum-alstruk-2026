## ANALISIS & PEMBAHASAN SOAL

### Soal 1 (Array & Struct)

Pada soal ini, saya belajar cara menggabungkan beberapa jenis data seperti nama buku (string) dan harga (float) ke dalam satu wadah bernama `struct Buku`. Data ini kemudian disimpan dalam sebuah `array` agar bisa menampung banyak buku sekaligus.

Saya menggunakan perulangan `for` untuk memasukkan data dan menampilkannya kembali dalam bentuk tabel. Untuk mencari harga tertinggi, saya membandingkan harga satu per satu. Sedangkan untuk rata-rata, semua harga dijumlahkan lalu dibagi dengan jumlah buku. Program ini sangat membantu untuk belajar dasar pengelolaan data kelompok.

### Soal 2 (Stack & Queue)

Di soal kedua, saya belajar tentang dua konsep penyimpanan data yang berbeda:
1. **Stack (Tumpukan)**: Menggunakan konsep LIFO (Last In First Out). Artinya, data yang terakhir masuk akan menjadi yang pertama keluar, seperti tumpukan struk belanja. Saya membuatnya manual memakai array dan variabel `top`.
2. **Queue (Antrian)**: Menggunakan konsep FIFO (First In First Out). Artinya, data yang pertama masuk akan dilayani duluan, persis seperti antrian kasir. Di sini saya menggunakan variabel `front` dan `rear` untuk menandai depan dan belakang antrian.

Kedua konsep ini dikerjakan secara manual tanpa bantuan library otomatis agar saya lebih paham bagaimana data masuk dan keluar dari memori.
