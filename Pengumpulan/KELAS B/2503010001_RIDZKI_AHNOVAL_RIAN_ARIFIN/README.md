## Soal 2

Pada soal kedua, program menggunakan konsep Stack dan Queue.

Stack digunakan untuk menyimpan riwayat transaksi dengan metode LIFO (Last In First Out), yaitu data terakhir yang masuk akan keluar lebih dahulu. Operasi yang digunakan yaitu push, pop, dan peek.

Queue digunakan untuk antrian pelanggan dengan metode FIFO (First In First Out), yaitu data pertama yang masuk akan keluar lebih dahulu. Operasi yang digunakan yaitu enqueue, dequeue, dan peek.

Konsep yang digunakan:
- Stack
- Queue
- Array
- Function
- Percabangan
