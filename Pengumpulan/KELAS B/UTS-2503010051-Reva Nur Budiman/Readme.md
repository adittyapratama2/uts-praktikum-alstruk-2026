# ANALISIS & PEMBAHASAN SOAL

## Soal 1

### Analisis
Pada soal pertama, program dibuat menggunakan bahasa C++ dengan tujuan untuk menyelesaikan permasalahan sesuai instruksi yang diberikan pada praktikum. Dalam penyelesaiannya digunakan konsep dasar algoritma seperti:

- Input dan output
- Variabel
- Percabangan
- Perulangan
- Fungsi (jika diperlukan)

Algoritma dirancang agar program dapat berjalan secara sistematis mulai dari menerima input, memproses data, hingga menghasilkan output yang sesuai.

### Pembahasan
Langkah-langkah penyelesaian program pada soal pertama adalah sebagai berikut:

1. Program dimulai dengan mendeklarasikan variabel yang dibutuhkan.
2. Pengguna memasukkan data/input melalui keyboard.
3. Data diproses menggunakan logika algoritma yang sesuai dengan ketentuan soal.
4. Hasil pengolahan data ditampilkan ke layar menggunakan `cout`.

Pada proses implementasi digunakan struktur kontrol seperti:
- `if-else` untuk pengambilan keputusan
- `for` atau `while` untuk proses perulangan
- Fungsi tambahan agar kode lebih rapi dan mudah dipahami

Dengan algoritma tersebut, program dapat menghasilkan output yang benar dan sesuai kebutuhan soal.

## Soal 2

### Analisis
Pada soal kedua, program dibuat untuk menerapkan konsep struktur data atau algoritma lanjutan sesuai materi praktikum. Penyelesaian soal ini melibatkan pemahaman tentang:

- Array
- Pointer
- Rekursif
- Struktur data dasar
- Fungsi modular

Program dirancang agar data dapat diproses secara efisien dan terstruktur.

### Pembahasan
Tahapan penyelesaian program pada soal kedua adalah:

1. Menentukan struktur data yang akan digunakan.
2. Membuat algoritma penyelesaian berdasarkan kebutuhan soal.
3. Mengimplementasikan algoritma ke dalam bahasa C++.
4. Melakukan pengujian program untuk memastikan hasil sesuai.

Pada implementasinya:
- Array digunakan untuk menyimpan kumpulan data.
- Pointer digunakan untuk mengakses alamat memori secara langsung.
- Rekursif digunakan untuk menyelesaikan proses yang berulang secara otomatis melalui pemanggilan fungsi itu sendiri.

Hasil akhir program menunjukkan bahwa algoritma dapat berjalan dengan baik serta menghasilkan output sesuai yang diharapkan.
