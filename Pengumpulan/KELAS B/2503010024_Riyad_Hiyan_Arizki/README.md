## ANALISIS & PEMBAHASAN SOAL

### Soal 1

Analisi
Program dibuat untuk mengelola data mahasiswa menggunakan Array of Struct. Struct digunakan karena data mahasiswa memiliki beberapa atribut dengan tipe data berbeda, seperti NIM, nama, semester, dan IPK.

Pembahasan
Pertanyaan A
Saya membuat struct Mahasiswa dengan atribut nim, nama, semester, dan ipk. Data disimpan dalam array berkapasitas 5 elemen. Input dilakukan menggunakan perulangan for, lalu data ditampilkan dalam bentuk tabel menggunakan setw() agar rapi.

Pertanyaan B
IPK tertinggi dicari menggunakan perulangan dan variabel idxMaks. Rata-rata IPK dihitung dari total seluruh IPK dibagi jumlah mahasiswa. Pencarian semester dilakukan dengan membandingkan input pengguna dengan data pada array.

### Soal 2

Analisis
Soal ini membahas implementasi Stack dan Queue secara manual menggunakan array statis tanpa STL.

Pembahasan
Pertanyaan A (Stack)
Stack menggunakan konsep LIFO (Last In, First Out). Implementasi menggunakan array berkapasitas 5 dan variabel top. Fungsi push() digunakan untuk menambah data, pop() untuk menghapus data teratas, dan peek() untuk melihat data paling atas. Program juga menggunakan pengecekan isFull() dan isEmpty().

Pertanyaan B (Queue)
Queue menggunakan konsep FIFO (First In, First Out). Implementasi menggunakan array serta variabel front dan rear. Fungsi enqueue() digunakan untuk menambah data, dequeue() untuk menghapus data depan, dan peek() untuk melihat data terdepan.
