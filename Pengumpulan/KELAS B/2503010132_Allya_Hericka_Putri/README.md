ANALISIS & PEMBAHASAN SOAL
Soal 1
Soal 1 berfokus pada penggunaan array of struct untuk mengelola data mahasiswa. Struct digunakan untuk mengelompokkan beberapa atribut dalam satu kesatuan data, yaitu NIM, nama, semester, dan IPK. Dengan menggunakan array, program mampu menyimpan lebih dari satu data mahasiswa dalam satu variabel terstruktur.

Pada tahap input, program meminta pengguna memasukkan jumlah mahasiswa (maksimal 5), kemudian mengisi data masing-masing mahasiswa secara berulang menggunakan perulangan. Hal ini menunjukkan penerapan konsep looping dan input data dinamis.

Soal 2
A. Stack (LIFO) Pada bagian stack, program mensimulasikan tumpukan buku dengan prinsip Last In First Out (LIFO), di mana data terakhir yang masuk akan menjadi yang pertama keluar. Struktur stack dikontrol menggunakan variabel top sebagai penunjuk posisi elemen teratas.

B. Queue (FIFO) Pada bagian queue, program mensimulasikan antrian peminjaman buku dengan prinsip First In First Out (FIFO), di mana data yang pertama masuk akan keluar lebih dulu. Queue menggunakan dua penunjuk: front → elemen depan rear → elemen belakang
