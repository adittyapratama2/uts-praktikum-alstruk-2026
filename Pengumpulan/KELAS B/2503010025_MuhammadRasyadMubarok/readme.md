## ANALISIS & PEMBAHASAN SOAL

### Soal 1

Pada soal pertama program dibuat menggunakan Array of Struct.
Struct digunakan untuk menyimpan beberapa data buku dalam satu
variabel seperti judul, pengarang, tahun terbit, dan harga.

Program dapat:
- Menginput data buku
- Menampilkan data dalam bentuk tabel
- Menampilkan buku dengan harga tertinggi
- Menghitung rata-rata harga buku
- Mencari buku berdasarkan tahun terbit

Perulangan `for` digunakan untuk input dan menampilkan data,
sedangkan `if` digunakan untuk proses pencarian dan menentukan
harga tertinggi.

### Soal 2

Pada soal kedua program menggunakan Stack dan Queue manual
menggunakan array tanpa STL.

Stack digunakan untuk riwayat transaksi:
- `push()` untuk menambah data
- `pop()` untuk menghapus data terakhir
- `peek()` untuk melihat data paling atas

Queue digunakan untuk antrian pelanggan:
- `enqueue()` untuk menambah antrian
- `dequeue()` untuk menghapus antrian depan
- `peek()` untuk melihat antrian paling depan

Program juga menggunakan fungsi `isEmpty()` dan `isFull()`
untuk mengecek kondisi stack dan queue.
