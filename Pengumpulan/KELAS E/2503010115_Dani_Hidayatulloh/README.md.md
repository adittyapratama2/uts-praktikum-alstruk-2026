## ANALISIS & PEMBAHASAN SOAL

### Soal 1

Program ini menggunakan struct Buku untuk menyimpan data judul, pengarang, tahun terbit, dan harga, lalu menyimpannya dalam array berkapasitas 5 data. Program menerima input sejumlah buku, menampilkan seluruh data dalam bentuk tabel, mencari buku dengan harga tertinggi, menghitung rata-rata harga, dan menampilkan buku berdasarkan tahun terbit yang dimasukkan pengguna.

### Soal 2

Program ini mengimplementasikan Stack dan Queue secara manual menggunakan array statis berkapasitas 5 elemen. Stack digunakan untuk menyimpan riwayat transaksi dengan prinsip LIFO melalui operasi push, pop, dan peek, sedangkan Queue digunakan untuk mengelola antrian pelanggan dengan prinsip FIFO melalui operasi enqueue, dequeue, dan peek. Program juga menampilkan isi data setelah setiap operasi serta pesan saat struktur data penuh atau kosong.
