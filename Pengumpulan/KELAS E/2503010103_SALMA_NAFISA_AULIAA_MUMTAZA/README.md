Analisis dan Pembahasan Soal
Soal 1:
Program pada soal 1 digunakan untuk menyimpan data buku menggunakan Array of Struct. Struct digunakan untuk menggabungkan data buku seperti judul, pengarang, tahun terbit, dan harga, sedangkan array digunakan untuk menyimpan beberapa data buku sekaligus dalam satu variabel. Program dapat melakukan input dan menampilkan data buku, mencari buku dengan harga tertinggi, menghitung rata-rata harga buku, serta mencari buku berdasarkan tahun terbit. Konsep yang digunakan dalam program ini yaitu struct, array, perulangan, dan percabangan.

Soal 2:
Program pada soal 2 membahas implementasi Stack dan Queue secara manual menggunakan array. Stack digunakan untuk menyimpan riwayat transaksi dengan konsep LIFO (Last In First Out), yaitu data terakhir masuk akan keluar terlebih dahulu. Sedangkan Queue digunakan untuk antrian pelanggan dengan konsep FIFO (First In First Out), yaitu data pertama masuk akan keluar terlebih dahulu. Program menggunakan fungsi push, pop, enqueue, dequeue, peek, isEmpty, dan isFull untuk mengelola data pada stack dan queue sehingga proses penyimpanan dan pengambilan data dapat berjalan dengan baik.
