## ANALISIS & PEMBAHASAN SOAL

### Soal 1

Program ini dapat menyimpan beberapa data buku yang terdiri dari judul, pengarang, tahun terbit, dan harga. Data dimasukkan oleh pengguna menggunakan perulangan for, kemudian seluruh data buku ditampilkan dalam bentuk tabel sederhana.Selain itu, program ini juga memiliki fitur untuk mencari buku dengan harga tertinggi, menghitung rata-rata harga seluruh buku, serta mencari buku berdasarkan tahun terbit tertentu. Dalam program digunakan konsep dasar algoritma dan struktur data seperti array, struct, looping, percabangan, dan input-output.

### Soal 2

Program ini menerapkan dua struktur data, yaitu Stack dan Queue.
Stack digunakan untuk menyimpan data transaksi dengan konsep LIFO (yang terakhir masuk akan keluar pertama). Di dalamnya ada fungsi untuk menambah data (push), menghapus data (pop), melihat data paling atas (peek), dan menampilkan seluruh isi stack.
Sedangkan Queue digunakan untuk menyimpan data pelanggan dengan konsep FIFO (yang pertama masuk akan keluar pertama). Di Queue juga ada fungsi untuk menambah data (enqueue), menghapus data (dequeue), melihat data paling depan, dan menampilkan seluruh antrian.Secara sederhana, program ini menunjukkan cara kerja dasar Stack dan Queue dalam mengelola data secara terstruktur.
