## ANALISIS & PEMBAHASAN SOAL

### Soal 1
**Analisis:**
Program ini menggunakan konsep **Array of Struct** untuk mengelola data mahasiswa. Struct `Mahasiswa` digunakan untuk mengelompokkan berbagai tipe data (string, int, float) menjadi satu entitas. Program mendukung input dinamis (maksimal 5 data) dan menampilkannya dalam format tabel.

**Pembahasan:**
- **Fitur Statistik**: Menggunakan algoritma pencarian nilai maksimum secara sekuensial untuk menentukan IPK tertinggi dan menghitung rata-rata dengan menjumlahkan seluruh IPK dibagi jumlah mahasiswa.
- **Fitur Filter**: Menggunakan seleksi kondisi `if` di dalam perulangan untuk mencari mahasiswa berdasarkan nomor semester tertentu.

### Soal 2
**Analisis:**
Program ini mengimplementasikan dua struktur data dasar, yaitu **Stack (Tumpukan)** dan **Queue (Antrean)** secara manual menggunakan array statis tanpa bantuan library STL.

**Pembahasan:**
- **Stack (LIFO)**: Digunakan untuk simulasi tumpukan buku kembali. Menggunakan variabel `top` sebagai indeks posisi paling atas. Operasi `push` menambah data ke tumpukan, dan `pop` mengambil data terakhir yang masuk.
- **Queue (FIFO)**: Digunakan untuk simulasi antrean peminjaman buku. Menggunakan variabel `front` untuk pintu keluar dan `rear` untuk pintu masuk. Implementasi menggunakan konsep *Circular Queue* (antrean melingkar) agar penggunaan memori array lebih efisien.
