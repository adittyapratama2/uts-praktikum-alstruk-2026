#Soal 1

Program pada soal 1 menggunakan struktur data struct dengan nama Mahasiswa yang berisi data Nim, Nama, Semester, dan IPK. 
Program meminta pengguna memasukkan jumlah data mahasiswa maksimal 5 data, kemudian data tersebut disimpan ke dalam array struct.

Setelah data dimasukkan, program akan menampilkan seluruh data mahasiswa dalam bentuk tabel. Selain itu program juga melakukan beberapa proses lain seperti:
1. Mencari IPK tertinggi dari seluruh data mahasiswa.
2. Menghitung rata-rata IPK mahasiswa.
3. Melakukan pencarian data berdasarkan semester yang diinputkan pengguna.

Pada bagian pencarian IPK tertinggi digunakan variabel indeks bernama Tinggi untuk menyimpan posisi mahasiswa dengan nilai IPK terbesar. 
Sedangkan untuk menghitung rata-rata IPK digunakan proses penjumlahan seluruh IPK lalu dibagi jumlah data mahasiswa.

Program ini menunjukkan penggunaan:
- Struct
- Array
- Perulangan for
- Percabangan if
- Variabel boolean
- Input dan output data

Kesimpulannya, program berhasil mengelola data mahasiswa sekaligus melakukan proses pencarian dan perhitungan sederhana.

#Soal 2

Program pada soal 2 membahas implementasi struktur data Stack dan Queue menggunakan array.

A. Stack
Stack menggunakan konsep LIFO (Last In First Out), yaitu data terakhir yang masuk akan keluar terlebih dahulu.
Pada program terdapat beberapa fungsi:
- push() untuk menambahkan data ke stack
- pop() untuk menghapus data teratas
- peekStack() untuk melihat data paling atas
- tampilStack() untuk menampilkan isi stack

Program memasukkan beberapa mata kuliah ke dalam stack seperti Fisika Dasar, Kalkulus, Algoritma, dan Jaringan. 
Kemudian program menampilkan isi stack dan melakukan proses penghapusan data.

B. Queue
Queue menggunakan konsep FIFO (First In First Out), yaitu data pertama yang masuk akan keluar terlebih dahulu.
Fungsi yang digunakan antara lain:
- enqueue() untuk menambah data
- dequeue() untuk menghapus data terdepan
- peekQueue() untuk melihat data terdepan
- tampilQueue() untuk menampilkan isi queue

Program memasukkan beberapa nama ke dalam queue seperti Reza, Mia, dan Hendra, lalu menampilkan isi queue dan melakukan proses dequeue.

Program ini menunjukkan pemahaman dasar mengenai:
- Struktur data Stack
- Struktur data Queue
- Array
- Fungsi/procedure
- Konsep LIFO dan FIFO
