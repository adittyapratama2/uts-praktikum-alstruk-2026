## ANALISIS & PEMBAHASAN SOAL

### Soal 1
**Analisis:** Menggunakan struktur data `struct` untuk mengelompokkan atribut mahasiswa (NIM, Nama, IPK). Data disimpan dalam `array` dan ditampilkan kembali menggunakan perulangan. Implementasi ini memudahkan pengelolaan data kelompok dalam satu variabel.

### Soal 2
**Analisis:**
- **Stack (Tumpukan):** Menerapkan prinsip LIFO (*Last In First Out*). Berguna untuk operasi yang membutuhkan urutan balik.
- **Queue (Antrian):** Menerapkan prinsip FIFO (*First In First Out*). Berguna untuk pemrosesan data berdasarkan urutan kedatangan.
